# File Structure

-  **`Project_Folder`**
     - `src` :
        - **index.js** : DB connects.
        - **app.js** : config, cookie, urlencode
        - **constants.js** : enums, DB-name
        - `DB` : Actual code to connect the db
        - `Models` : Data schema
        - `Controllers` : Functionalities or Methods
        - `Routes` : Handling the routing
        - `Middlewares`
        - `Utils` : Utilities
    - `package.json`
    - `.env`
    - `Readme.md`
